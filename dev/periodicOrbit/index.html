Periodic orbits computation

Consider the DDE problem written

\[\frac{du}{dt}=F(u(t), u_t;p).\tag{E}\]

A periodic solution $u^*$ with period $T$ satisfies $u^*(t+T)=u^*(t)$ for all time $t$.
